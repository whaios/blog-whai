Visual Studio Code 快捷键

发布时间 : 2018-07-03
发布 : 2018-07-03 分类 : Notes 浏览 :

VsCode 快捷键有五种组合方式（科普）

  1. Ctrl + Shift + ? : 这种常规组合按钮
  2. Ctrl + C Ctrl +V : 同时依赖一个按键的组合
  3. Shift + V C : 先组合后单键的输入
  4. Ctrl + Click: 键盘 + 鼠标点击
  5. Ctrl + DragMouse : 键盘 + 鼠标拖动
