Markdown 新手指南

发布时间 : 2018-02-07
发布 : 2018-02-07 分类 : Notes 浏览 :

Markdown 是一种「电子邮件」风格的「标记语言」。
使用它编写文章具有以下优点：

  • 纯文本，所以兼容性极强，可以用所有文本编辑器打开。
  • 让你专注于文字而不是排版。
  • 格式转换方便，Markdown 的文本你可以轻松转换为 html、电子书等。
  • Markdown 的标记语法有极好的可读性。
