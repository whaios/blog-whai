使用 Ghost 搭建个人 Wiki

发布 : 2018-03-29 分类 : Notes 浏览 :

wwiki theme for Ghost

用于搭建个人 Wiki 的 Ghost 主题，主题样式参照 gitbook 系统。

主题特点

  • 适用于搭建 个人wiki、项目文档、API文档类网站
  • 参照 gitbook 系统，版面整洁，段落样式清晰美观
  • 前端可调整字号、字体 和 阅读主题
  • 自适应，支持手机阅读
  • 文章（标题）搜索功能，支持中文搜索

如何使用

  1. 部署 Ghost 博客，不会的请参考《部署Ghost1.20+博客》
  2. 下载主题包 wwiki ，
  3. 通过 Ghost 后台的 Design -> Upload a theme 上传主题并启用
  4. 完成

主题说明

  • 左侧目录树：通过 Tag 对文章进行分类，没有 Tag 的文章显示在目录树最下边
  • 首页：Post Url 为 index 的文章将显示到首页

模版下载

主题模版托管在Github whaios/wwiki
